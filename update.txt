 {59EF5B}## 2.5.18 | {F3D735}7 апреля 2020
1. Переделан механизм биндера. Теперь можно создавать диалоги внутри диалога. Тем самым 
расширены возможности биндинга.
Более подробно в разделе "Биндер" -> "Для продвинутых".
2. Добавлена проверка на ошибки диалогов в биндере.
3. Добавлен тег для биндера: {myHospEn}. Возвращает анг. полное название больницы.
4. Добавлен тег-функция для биндера: {sleep:время}. Создаёт свою задержку между строчками. 
Время задаётся в мс (1 сек = 1000 мс)
5. Добавлен тег-функция для биндера: {sex:текст1|текст2}. Возвращает текст в зависимости 
от выбранного пола.
текст1 - мужской, текст2 - женский. 
6. Добавлен тег-функция для биндера: {getNickByID:ид игрока}. Возвращает ник игрока по его ID.
Может комбинироваться с функциями, которые возвращают ID игрока.
7. Добавлена система переменных для биндера. Теперь можно создавать переменные по форме 
#имя переменной=значение. 
Имя переменной может содержать только английские символы и цифры. Значение может быть любым.
Более подробно в разделе "Биндер" -> Система диалогов.
8. Добавлена возможность комментировать строчки с помощью двойного слеша //.
В комментариях можно оставлять для себя пометки. Комментарий не будет виден в чате.
9. Пустые строчки не будут включаться в бинд, тем самым можете разделять текст для себя.
10. Теперь отыгровку мед.карты можно самостоятельно менять по системе биндера.
Изменить можно в разделе "Настройки" -> "Отыгровки"
11. Убраны со многих команд ограничения по рангу.
12. Введена статистика по пользованию скрипта. По возможности в будущем будет публична.
13. Fix багов с меню департамента
14. Убрана защита по времени с подачей собеседования через меню департамента.
15. Небольшие изменения с рендером текста при отыгровках.
